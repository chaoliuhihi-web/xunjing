更新时间：2026-06-21

## 当前上线形态

当前版本是静态 React 官网，适合部署到 Nginx、宝塔静态站点、对象存储静态网站、CDN 源站或后续 Gitee Pages。页面表单目前为前端成功态，尚未接入真实 CRM/API。

## Nginx 静态部署

建议使用版本化发布目录：

```text
/www/wwwroot/xinghexunjing/
  releases/
    20260621-010000/
      index.html
      assets/
      favicon.svg
  current -> releases/20260621-010000
```

Nginx 站点配置：

```nginx
server {
    listen 80;
    server_name your-domain.example;

    root /www/wwwroot/xinghexunjing/current;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    location ^~ /assets/ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

发布步骤：

```bash
npm ci
npm run test:run
npm audit --audit-level=moderate
npm run build
tar -czf xinghexunjing-dist.tar.gz dist
```

服务器上解压到新的 release 目录后切换软链：

```bash
ln -sfn /www/wwwroot/xinghexunjing/releases/YYYYMMDD-HHMMSS /www/wwwroot/xinghexunjing/current
nginx -t
nginx -s reload
```
