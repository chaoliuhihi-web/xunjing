# 星河寻境后台环境变量清单

更新时间：2026-06-21

## 1. 原则

- 真实密钥只允许写入本机未提交的 `.env.local`、服务器密钥管理或 CI/CD Secret。
- 文档、示例文件和代码默认只保留变量名、用途和默认空值。
- 星河寻境作为独立项目使用独立数据库、缓存前缀、对象存储路径和向量库 collection。

## 2. 基础运行

| 变量 | 用途 | P0 默认值 |
| --- | --- | --- |
| `SPRING_PROFILES_ACTIVE` | Spring 运行环境 | `local` |
| `XUNJING_TENANT_ID` | Yudao 多租户请求头租户编号；小程序和 live 门禁必须传 `tenant-id` | 喀什 P0 seed 为 `1` |
| `INTERNAL_AUTH_TOKEN` | 内部导入、爬虫回调、批处理接口鉴权 | 本机或服务器 Secret |

## 9. 上线检查

- `.env.local` 不允许进入 git。
- 生产环境必须使用 `yudao_xinghe_xunjing` 或同等独立库名。
- 生产环境 Redis 必须使用独立 DB 或 `xj:` 前缀。
- 生产环境 Qdrant collection 不允许与 XingheAI2026V2 共用。
- 导入、爬虫、公益报告生成等内部接口必须校验 `INTERNAL_AUTH_TOKEN` 或 Yudao 后台权限。
