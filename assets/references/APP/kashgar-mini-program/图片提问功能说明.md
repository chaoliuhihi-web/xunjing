# 图片提问功能实现说明

## 当前功能边界

### 1. 核心功能
- 点击图片按钮选择图片。
- 在聊天中保留所选图片气泡。
- 显示图片识别能力需要后端开通的提示。
- 不在 APP 客户端直连第三方文件上传接口。
- 不在 APP 客户端保存 Coze Token 或 PAT。

### 2. 技术实现

#### 当前流程
```
用户点击图片按钮
    ↓
选择图片 (uni.chooseImage)
    ↓
聊天区显示图片
    ↓
提示用户先用文字描述图片
    ↓
等待后端文件识别代理接入后再打开真正图片问答
```

#### 关键代码

**图片进入聊天区**
```javascript
messages.value.push(createUserMessage({
  content: '',
  images: [filePath]
}))
```

**当前提示**
```javascript
commitAssistantMessage(assistantMessage, {
  isPending: false,
  content: '图片问答需要后台开通文件识别能力，请先用文字描述图片内容，我可以继续帮你解读喀什景点、路线和故事。'
})
```

**文本 AI 问答**
```javascript
const aiResult = await startXunjingAiRequest({
  question: userMessage,
  assistantMessage
})
```

## 测试步骤

### 场景 1：选择图片
1. 打开 AI 小导游页面。
2. 点击图片图标。
3. 选择一张图片。
4. 观察用户消息显示所选图片，AI 消息提示先用文字描述图片，且不产生第三方文件上传请求。

### 场景 2：文本提问
1. 在输入框输入文字。
2. 点击发送。
3. 确认 AI 对话通过 `/app-api/xunjing/ai/chat` 返回。

## 后续接入要求

1. 图片识别必须新增后端代理接口，例如 `/app-api/xunjing/ai/image-chat`。
2. 后端代理负责保存和调用第三方密钥，APP 只上传业务参数和文件。
3. APP 请求必须携带 `tenant-id`。
4. 接通后再恢复纯图片提问、文字加图片、多图识别和上传进度。
